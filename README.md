# DevNexus (Developer Toolbox)

DevNexus is a plugin-based desktop toolbox built with Tauri + React.
Phase 1 targets a Redis management plugin.

## Development

- `npm run dev`: start front-end dev server
- `npm run build`: build front-end assets
- `npm test`: run Vitest test suite
- `npm run lint`: run TypeScript static checks

## Tauri Runtime

To run the full desktop app you also need Rust and Tauri prerequisites:
- https://www.rust-lang.org/tools/install
- https://tauri.app/start/prerequisites/
