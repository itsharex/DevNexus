# DevNexus v0.9.3

## Highlights

- Publishes separate macOS DMG assets for Intel (`x64`) and Apple Silicon (`arm64`) in CI and tag-triggered releases.
- Restores the native macOS title bar and red/yellow/green traffic-light window controls on macOS.
- Fixes the macOS/root layout height so the main app, sidebar, status bar, and bottom-left Chat launcher fill the full window without leaving a large blank area.
- Fixes dark-theme background coverage at the bottom of the app window.
- Improves LAN Chat dark-theme styling across the floating window, session list, composer, message bubbles, and pane dividers.
- Prevents the LAN Chat nickname setup input from being reset by snapshot refreshes while the user is typing.
- Uses the macOS network interface MAC address for LAN Chat stable Device ID generation, with existing UUID identities migrating on startup.

## Validation

- `npm test`
- `npm run build`
- `cd src-tauri && cargo check`
- `npm run tauri build -- --target x86_64-apple-darwin --bundles app`
- Local Intel DMG generated with `hdiutil create` and verified with `hdiutil verify`
