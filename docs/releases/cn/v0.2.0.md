# DevNexus v0.2.0

本次版本聚焦 Redis 使用体验升级，并延续跨平台发布能力。

## 亮点

- Redis Key Browser 交互升级：Key Tree 和 Key Detail 支持可拖拽分栏。
- 分栏支持最小宽度保护，避免拖拽后内容不可见。
- 持续提供 Windows、macOS、Linux 三平台安装包。

## Redis 改进

- 在 Key Tree 与 Key Detail 之间新增可拖动分割条。
- 拖动时实时调整左右面板宽度。
- 分割条悬浮与拖动状态高亮，交互反馈更清晰。
- 左侧最小宽度：`300px`。
- 右侧最小宽度：`420px`。

## 安装包

- Windows：NSIS 安装包（`.exe`）。
- macOS：App Bundle（`.app`）和安装包（`.dmg`）。
- Linux：Debian 包（`.deb`）和 AppImage（`.AppImage`）。

## 说明

- Release 资产由 GitHub Actions 在推送 `v0.2.0` tag 后自动构建并上传。
- 后续版本可继续补充分栏宽度本地持久化，让重启后保持上次布局。
