# DevNexus v0.6.0 发布说明

## 亮点

- 新增 Network 诊断插件，用于一次性连接排障。
- 新增 TCP 端口检测，并支持超时控制。
- 新增 Ping 诊断，解析包数量、丢包率、延迟摘要和原始输出。
- 新增 DNS A/AAAA 查询支持。
- 新增 Traceroute 诊断，支持 hop 解析和原始输出。
- 新增本地诊断历史，支持详情查看、复跑、删除和清空。

## 后续修复

- 修复 Windows Ping 输出解码和统计解析问题。
- 运行 Ping 和 Traceroute 时隐藏 Windows 控制台黑窗口。
- 降低 Traceroute 默认 hop/timeout，并增加全局等待上限，避免长时间无响应。
- 从 Network Tools 移除 HTTP 探测；HTTP/API 调试将作为独立 Postman-like 工具后续实现。
