# DevNexus v0.8.0

## 亮点

- 新增统一 MQ Client 插件，支持 RabbitMQ 和 Kafka。
- 新增 RabbitMQ 连接管理、Management API 浏览、publish、安全消费预览、消息模板和历史记录。
- 新增 Kafka 连接管理、metadata 浏览、produce、非提交 offset 的临时 consume 预览、消息模板和历史记录。
- 新增 MQ secret 加密和历史脱敏，覆盖密码、Management 凭据、SASL secret 和敏感 headers。

## 安全边界

- RabbitMQ 预览消费默认 requeue，除非显式选择 Ack。
- Kafka 预览消费不会提交 offset。
- 首个 MQ 版本不暴露 queue purge/delete、exchange delete、topic create/delete、offset commit、broker/topic 配置修改等破坏性操作。
