# DevNexus v0.1.0

首个正式发布版本，完成 DevNexus 工具箱基础能力与三大插件体系落地。

## 亮点

- 全新品牌升级为 DevNexus。
- 落地插件化桌面架构，基于 Tauri 2、React 和 Rust。
- 增加 Windows、macOS、Linux 跨平台构建与发布流程。

## 包含插件

### Redis Manager

- 连接管理：新增、编辑、删除、测试和连接。
- Key 浏览：扫描、筛选、分页和批量操作。
- 支持 String、Hash、List、Set、ZSet 五种数据类型编辑。
- 命令控制台执行与查询历史。
- Server 信息监控和基础图表。
- 支持 JSON / CSV 导入导出。

### SSH Client

- SSH 连接管理。
- 多标签终端会话。
- 密钥管理。
- 隧道管理。

### S3 Client 基础版

- S3 连接管理。
- Bucket 浏览，支持创建、删除和刷新。
- Object 浏览，支持前缀导航、加载更多、新建文件夹和删除。
- 连接后自动切换到 Buckets，打开 Bucket 后自动切换到 Objects。

## 安装包

- Windows：NSIS 安装包（`.exe`）。
- macOS：App Bundle（`.app`）和安装包（`.dmg`）。
- Linux：Debian 包（`.deb`）和 AppImage（`.AppImage`）。

## 说明

- macOS 包如需分发给非本机用户，后续建议补充 Apple 证书签名和 notarization。
- 当前版本聚焦可用主链路；更高级的 S3 功能如上传下载队列、预览、预签名 URL、生命周期策略将在后续版本继续推进。
