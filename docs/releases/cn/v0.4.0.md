# DevNexus v0.4.0

## 亮点

- 新增 MongoDB 客户端插件。
- 支持 MongoDB URI 和表单式连接配置，并对本地敏感信息加密存储。
- 新增数据库与集合浏览器。
- 新增文档浏览、插入、更新、删除、过滤、投影、排序和分页。
- 新增聚合与数据库命令工作区，并支持查询历史。
- 新增索引列表、创建和删除管理。
- 新增 JSON / JSONL 导入导出。
- 新增 MongoDB Server Status 视图。

## 验证

- `cargo check`
- `npm test`
- `npm run build`
- `npm run tauri build -- --bundles nsis`
