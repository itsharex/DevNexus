# DevNexus v0.7.1

## 亮点

- 基于 issue #1 反馈优化 SSH terminal 工作区。
- 新增单个 SSH terminal tab 的全屏模式。
- 增加 xterm 底部 padding，避免当前命令行紧贴底边。
- 输入时新增命令建议，结合连接快捷命令和常用 shell 命令。

## 体验说明

- 输入至少两个字符后显示命令建议。
- 点击建议只补全当前命令草稿，不会自动执行。
- 全屏模式会重新适配 xterm 视口，可从 terminal 工具栏退出。

## 验证

- `npm test`
- `npm run build`
- `cd src-tauri && cargo check`
- `npm run tauri build -- --bundles nsis`
