# DevNexus v0.5.0

## 亮点

- 新增 MySQL 客户端插件。
- 支持加密的 MySQL 连接配置和连接测试。
- 新增数据库/表浏览器，支持表字段和状态摘要。
- 新增分页表数据浏览，并支持基于主键的插入、更新和删除。
- 新增 SQL 工作区，支持表格/JSON 结果视图和查询历史。
- 新增索引列表、创建和删除管理。
- 新增 JSON / CSV 导入导出。
- 新增 MySQL Server Status 视图。

## 验证

- `cargo check`
- `npm test`
- `npm run build`
- `npm run tauri build -- --bundles nsis`
