# DevNexus v0.9.3

## 亮点

- CI 和 tag 触发发布中分别产出 Intel (`x64`) 与 Apple Silicon (`arm64`) 两套 macOS DMG。
- macOS 恢复原生标题栏和红黄绿 traffic-light 窗口控制按钮。
- 修复 macOS/root 布局高度问题，让主应用、侧边栏、状态栏和左下角 Chat 入口铺满整个窗口，避免大块空白区域。
- 修复应用窗口底部暗色主题背景覆盖问题。
- 优化 LAN Chat 悬浮窗、会话列表、输入区、消息气泡和面板分割线的暗色主题样式。
- 修复 LAN Chat 昵称设置输入框在用户输入时被 snapshot 刷新重置的问题。
- LAN Chat 稳定 Device ID 改为使用 macOS 网络接口 MAC 地址生成，启动时迁移已有 UUID 身份。

## 验证

- `npm test`
- `npm run build`
- `cd src-tauri && cargo check`
- `npm run tauri build -- --target x86_64-apple-darwin --bundles app`
- 本地 Intel DMG 通过 `hdiutil create` 生成，并通过 `hdiutil verify` 校验。
