# DevNexus v0.7.0

## 亮点

- 新增首版 API Debugger 插件，在 DevNexus 内提供轻量 Postman-like HTTP/API 工作区。
- 请求构建器支持 method、URL、query params、headers、cookies、auth、body、timeout、redirect 和 SSL 校验。
- 新增本地集合、文件夹、保存请求、环境变量和请求历史，数据存储在 SQLite 中。
- 支持 `{{variable}}` 环境变量替换和缺失变量预览。
- 响应查看支持状态码、耗时、大小、Content-Type、headers、cookies、Pretty JSON、Raw body 和 timing summary。
- 支持 cURL 导入和 DevNexus API Collection JSON 脱敏导出。

## 安全

- Secret 环境变量使用现有 AES-GCM helper 加密后再持久化。
- 历史快照默认脱敏 Authorization、Cookie、Set-Cookie，以及 token/password/key/secret 类字段。
- 大响应体在内存中截断并标记 truncated。

## 范围说明

- 本版本不追求完整兼容 Postman Collection v2.1。
- WebSocket、gRPC、Mock Server、团队同步、脚本/测试 Runner 保留为后续能力。
