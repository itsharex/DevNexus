# DevNexus v0.1.0

首个正式发布版本，完成 DevNexus 工具箱基础能力与三大插件体系落地。

## Highlights

- 全新品牌升级：RDMM -> DevNexus
- 插件化桌面架构（Tauri 2 + React + Rust）
- 跨平台构建与发布流程（Windows / macOS / Linux）

## Included Plugins

### Redis Manager

- 连接管理（新增/编辑/删除/测试/连接）
- Keys 浏览（扫描、筛选、分页、批量操作）
- 五种数据类型编辑（String / Hash / List / Set / ZSet）
- 控制台命令执行与历史记录
- 服务器信息监控与基础图表
- 导入导出能力（JSON / CSV）

### SSH Client

- SSH 连接管理
- Terminal 多标签会话
- Key 密钥管理
- 隧道管理（Tunnels）

### S3 Client（基础版）

- S3 连接管理
- Buckets 浏览（创建/删除/刷新）
- Objects 浏览（前缀导航、加载更多、新建文件夹、删除）
- 连接后自动跳转 Buckets，打开 Bucket 自动跳转 Objects

## Packages

- Windows: NSIS Installer (`.exe`)
- macOS: App Bundle (`.app`) + Installer (`.dmg`)
- Linux: Debian Package (`.deb`) + AppImage (`.AppImage`)

## Notes

- macOS 包若需分发给非本机用户，建议后续补充 Apple 证书签名与 notarization。
- 当前版本聚焦可用性主链路；更高级的 S3 功能（上传下载队列、预览、预签名、生命周期策略）将在后续版本推进。
