# DevNexus v0.3.0

本次版本完成第三期 S3 文件存储浏览器的主要功能闭环。

## Highlights

- S3 Client 从基础浏览升级为可用的对象管理工具
- 新增 Bucket 设置、对象详情、对象标签、预览、预签名 URL
- 新增对象上传/下载、批量删除、复制、移动、重命名等后端能力

## S3 Client

- 连接管理：新增、编辑、测试、连接后自动进入 Bucket
- Bucket 管理：列表、创建、删除、查看设置
- Bucket 设置：概览统计、Region、版本控制、Bucket Policy
- 对象浏览：面包屑路径、列表/网格视图、搜索、排序、加载更多
- 对象操作：预览、详情、标签编辑、删除、批量删除、重命名、复制路径、预签名 URL
- 传输能力：本地路径上传文件/文件夹，下载对象到本地路径
- 预览能力：文本内容直接预览，图片/二进制通过预签名 URL 预览或访问

## Packages

- Windows: NSIS Installer (`.exe`)
- macOS: Installer (`.dmg`)
- Linux: Debian Package (`.deb`) + AppImage (`.AppImage`)

## Notes

- 上传/下载当前使用本地路径输入，后续可接入 Tauri dialog 插件提供原生文件选择器。
- 超大文件断点续传和完整进度事件将在后续版本继续增强。
