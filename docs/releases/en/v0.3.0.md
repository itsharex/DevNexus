# DevNexus v0.3.0

This release completes the main feature loop for the Phase 3 S3 object-storage browser.

## Highlights

- Upgraded S3 Client from basic browsing to a practical object-management tool.
- Added bucket settings, object details, object tags, preview, and presigned URLs.
- Added backend support for object upload/download, batch delete, copy, move, and rename.

## S3 Client

- Connection management: create, edit, test, and automatically enter Buckets after connecting.
- Bucket management: list, create, delete, and view settings.
- Bucket settings: overview statistics, region, versioning, and bucket policy.
- Object browsing: breadcrumb navigation, list/grid views, search, sorting, and load more.
- Object actions: preview, details, tag editing, delete, batch delete, rename, copy path, and presigned URL.
- Transfer capabilities: upload local files/folders and download objects to local paths.
- Preview capabilities: preview text directly and access images/binary objects through presigned URLs.

## Packages

- Windows: NSIS installer (`.exe`).
- macOS: installer (`.dmg`).
- Linux: Debian package (`.deb`) and AppImage (`.AppImage`).

## Notes

- Upload/download currently use local path input. A future version can integrate the Tauri dialog plugin for native file pickers.
- Large-file resumable transfer and full progress events will continue to improve in later versions.
