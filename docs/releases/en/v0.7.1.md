# DevNexus v0.7.1

## Highlights

- Improved the SSH terminal workspace based on issue #1 feedback.
- Added a fullscreen mode for individual SSH terminal tabs.
- Added extra bottom padding inside xterm so the active command line no longer sits tight against the lower edge.
- Added command suggestions while typing, combining connection quick commands with common shell commands.

## UX Notes

- Command suggestions appear after at least two typed characters.
- Clicking a suggestion completes the current command draft without executing it automatically.
- Fullscreen mode re-fits the xterm viewport and can be exited from the terminal toolbar.
