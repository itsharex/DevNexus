# DevNexus v0.6.0 Release Notes

## Highlights

- Added the Network diagnostics plugin for one-shot connectivity troubleshooting.
- Added TCP port checks with timeout control.
- Added Ping diagnostics with parsed packet/loss/latency summary and raw output.
- Added DNS A/AAAA lookup support.
- Added Traceroute diagnostics with hop parsing and raw output.
- Added local diagnostic history with detail view, rerun, delete, and clear actions.

## Follow-up Fixes

- Fixed Windows Ping output decoding and statistics parsing.
- Hid Windows console windows when running Ping and Traceroute.
- Reduced Traceroute default hop/timeout settings and added a bounded global wait to avoid long unresponsive runs.
- Removed HTTP probing from Network Tools; HTTP/API debugging will be implemented later as a dedicated Postman-like tool.

## Validation

- `npm test`
- `npm run build`
- `cd src-tauri && cargo check`
- `npm run tauri build -- --bundles nsis`
