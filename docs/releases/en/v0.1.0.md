# DevNexus v0.1.0

The first official release, delivering the DevNexus toolbox foundation and the initial three-plugin product line.

## Highlights

- Rebranded the project to DevNexus.
- Added the plugin-based desktop architecture built with Tauri 2, React, and Rust.
- Added cross-platform build and release workflows for Windows, macOS, and Linux.

## Included Plugins

### Redis Manager

- Connection management: create, edit, delete, test, and connect.
- Key browsing with scanning, filtering, pagination, and batch operations.
- Editors for String, Hash, List, Set, and ZSet data types.
- Command console execution with query history.
- Server information monitoring and basic charts.
- Import and export support for JSON and CSV.

### SSH Client

- SSH connection management.
- Multi-tab terminal sessions.
- Key management.
- Tunnel management.

### S3 Client Basic

- S3 connection management.
- Bucket browsing with create, delete, and refresh actions.
- Object browsing with prefix navigation, load more, folder creation, and deletion.
- Automatically switches to Buckets after connecting and to Objects after opening a bucket.

## Packages

- Windows: NSIS installer (`.exe`).
- macOS: app bundle (`.app`) and installer (`.dmg`).
- Linux: Debian package (`.deb`) and AppImage (`.AppImage`).

## Notes

- macOS packages should later add Apple code signing and notarization before distribution to other users.
- This release focuses on the main usable workflow. Advanced S3 features such as upload/download queues, previews, presigned URLs, and lifecycle policies will continue in later releases.
