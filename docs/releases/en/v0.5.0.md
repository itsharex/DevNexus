# DevNexus v0.5.0

## Highlights

- Added MySQL client plugin.
- Supports encrypted MySQL connection profiles and connection testing.
- Added database/table browser with table columns and status summary.
- Added paginated table data browsing with primary-key based insert/update/delete.
- Added SQL workspace with table/JSON result views and query history.
- Added index list/create/drop management.
- Added JSON/CSV import and export.
- Added MySQL server status view.

## Verification

- `cargo check`
- `npm test`
- `npm run build`
- `npm run tauri build -- --bundles nsis`
