# DevNexus v0.2.0

This release focuses on Redis usability improvements while keeping cross-platform release support in place.

## Highlights

- Improved the Redis Key Browser interaction: Key Tree and Key Detail now support a draggable split boundary.
- Added minimum-width protection so panes remain usable after resizing.
- Continued to publish Windows, macOS, and Linux installation packages.

## Redis Improvements

- Added a draggable divider between Key Tree and Key Detail.
- Resizes left and right panes in real time while dragging.
- Adds hover and active states to make resizing feedback clearer.
- Left pane minimum width: `300px`.
- Right pane minimum width: `420px`.

## Packages

- Windows: NSIS installer (`.exe`).
- macOS: app bundle (`.app`) and installer (`.dmg`).
- Linux: Debian package (`.deb`) and AppImage (`.AppImage`).

## Notes

- Release assets are built and uploaded by GitHub Actions after pushing the `v0.2.0` tag.
- A future version may persist split-pane width locally so the layout is restored after restart.
