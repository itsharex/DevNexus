# DevNexus v0.4.0

## Highlights

- Added MongoDB client plugin.
- Supports MongoDB URI and form-based connection profiles with encrypted local secret storage.
- Added database and collection browser.
- Added document browsing, insert, update, delete, filter, projection, sort, and pagination.
- Added aggregate and database command workspace with query history.
- Added index list/create/drop management.
- Added JSON/JSONL import and export.
- Added MongoDB server status view.

## Verification

- `cargo check`
- `npm test`
- `npm run build`
- `npm run tauri build -- --bundles nsis`
