# DevNexus v0.7.0

## Highlights

- Added the first API Debugger plugin, a lightweight Postman-like HTTP/API workspace inside DevNexus.
- Added request builder support for method, URL, query params, headers, cookies, auth, body, timeout, redirects, and SSL validation.
- Added local collections, folders, saved requests, environments, and request history backed by SQLite.
- Added environment variable substitution with `{{variable}}` syntax and missing-variable preview.
- Added response inspection for status, duration, size, content type, headers, cookies, pretty JSON body, raw response, and timing summary.
- Added cURL import and DevNexus API collection JSON export with redaction support.

## Security

- Secret environment variables are encrypted with the existing AES-GCM helper before persistence.
- History snapshots redact Authorization, Cookie, Set-Cookie, token/password/key/secret-like fields by default.
- Large response bodies are capped in memory and marked as truncated.

## Scope Notes

- This release does not attempt full Postman Collection v2.1 compatibility.
- WebSocket, gRPC, mock server, team sync, and script/test runner features remain future work.
